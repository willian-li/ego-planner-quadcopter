# ego-planner-swarm

基于ROS的四旋翼无人机快递投放项目。主要功能：

- 三维空间避障：在空中遇到障碍物可以绕过,越过或者从下面穿过。
- 运载快递：可以搭载两个快递盒实现自主投放。
- 地标识别：识别地面标识选择是否投放货物，并根据地表位置实现精准投放。
